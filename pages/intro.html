Introduction

In this evaluation, you will be evaluating two different three dimensional user interfaces (3DUIs). You will be doing this by completing tasks which will be presented to you.

Please know that in this evaluation we are testing the interface, not you. If you get stuck and are not sure of how to proceed, please ask the evaluation coordinator for help. If you would like to end the experiment at anytime, you are welcome to do so.

Please complete each task you are given accurately and quickly.

Evaluation Procedure
  1. You will first complete a Mental Rotations Test (MRT). The instructions for this are on the next page.
  2. You will then be asked to complete some tasks. The documentation is presented in a booklet in front of you and will be accessible on each page under "Help".
  3. After you have completed a set of tasks, you will be asked to evaluate the 3DUI's usability using a SUS questionnaire.
If you need to refer back to anything read above, you can consult the hard copy of the manual.
